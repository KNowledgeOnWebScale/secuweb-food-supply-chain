## firefly deploy ethereum

Deploy a compiled solidity contract

### Synopsis

Deploy a solidity contract compiled with solc to the blockchain used by a FireFly stack. If the
contract has a constructor that takes arguments specify them as arguments to the command after the filename.

To compile a .sol file to a .json file run:

solc --combined-json abi,bin contract.sol > contract.json


```
firefly deploy ethereum <stack_name> <contract_json_file> [constructor_param1 [constructor_param2 ...]] [flags]
```
